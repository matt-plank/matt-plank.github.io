# matt-plank.github.io

This is my personal website. Built with React, Vite and TailwindCSS, deployed through GitHub Pages. Contains my contact information, portfolio of projects, and my technical blog.

## Setting Up

Clone the repository

```bash
$ git clone https://github.com/matt-plank/matt-plank.github.io.git
```

Install dependencies

```bash
$ cd matt-plank.github.io
$ npm install
```

## Running Locally for Development

```bash
# In the root directory
$ npm run dev
```

## Building for Production

```bash
# In the root directory
$ npm run build
```
